## Device Page

One of my main tasks was the designing and developing the **Device Management Page**, a central tool for installers to view, configure, and manage all hardware connected to a site. I implemented search, filtering, and sorting options so installers can quickly locate specific hardware across even multiple large installations. Once the backend API was finalized, I integrated real device data and aligned the UI with the correct API structure.

## Autodetection Script (Golang)

In support of onboarding, I've developed an autodetection script in Golang that scans the local network for compatible devices using both MAC and IP address discovery. This script executes ARP queries and response analysis to enumerate nearby hardware, detects vendor prefixes, and associated metadata like IP address, device type, and firmware.
On the frontend, I prepared the Device Page and modal components that would be used to display and add detected devices directly into the dashboard in future updates. The auto-detection script runs on the backend as a service, which lists the discovered devices for installers to swiftly review and configure.
Put together, the Device Page and the autodetection system comprise an efficient workflow that greatly enhances the way installers onboard and manage Harmony Energy devices.
